# DjangoBookStore

## How to start

```sh
git clone https://github.com/appletime81/DjangoBookStore.git
cd DjangoBookStore
```

then you need to install some dependencies.

```sh
pip install -r requirements.txt
```
type `python manage.py runserver` to start server.
